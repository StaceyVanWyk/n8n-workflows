# n8n Workflow Portfolio

## 🚀 Project: Daily Task Reminder

This project is a simple automation workflow built using n8n.

## 🧠 What it does

* Runs daily at a scheduled time
* Processes a list of tasks
* Converts them into structured data

## ⚙️ Workflow

Schedule Trigger → Set → Function

## 🛠 Tech Used

* n8n (Docker)
* JavaScript (Function Node)

## 💡 What I Learned

* Building automation workflows
* Data transformation using JavaScript
* Debugging real-world issues (Docker, UI bugs, JSON errors)
